## Getting Started

Run a local version of MongoDB with the following env vars set:
```
MONGO_INITDB_ROOT_USERNAME
MONGO_INITDB_ROOT_PASSWORD
```

To run the development server:

```bash
cp .env.example .env
# Be sure to fill in the `.env` file after copying
npm run dev
```
